1.  Проходим по ссылке https://github.com/ArtemKazaryan/Project .
2.  Далее жмём "Code" и в откроющейся вкладке "Local" нажимаем "Download ZIP".
3.  Нажимаем "Открыть" после чего в окне появится папка "Project-master", 
    открываем её и вложенную папку "PersonalMoneyTransaction" копируем на рабочий стол или в требуемую Вам дииректорию.
4.  Для открытия проекта папку "PersonalMoneyTransaction" открываем с помощью PyCharm. Далее нажимаем "Trust Project". 
5.  Для запуска проекта в терминале прописываем команду "python manage.py runserver" или открываем "Current File", 
    выбираем "Edit Configurations", нажимаем "+" затем нажимаем "Python" и в поле "Script path" указываем путь до файла "manage.py", 
    а в поле "Parameters" прописываем команду "runserver" и нажимаем "Ok".
6.  Зарегистрируйте пользователя введя логин пароль в поля регистрации. После этого только администратор вручную через админку 
    может сделать активным уже созданный аккаунт, поставив галочку в поле "Активный" прав доступа пользователя. 
7.  Для входа в админку или аккаунт администратора локальной версии приложения введите в адресную строку "http://127.0.0.1:8000/admin" и нажмите Enter.
8.  В появившееся окно введите логин: admin и пароль: Qq12345!
